# Nalora Kerala Calm

A modern mental wellness portal designed to connect patients with clinicians, featuring AI wellness chat support, online appointment bookings, patient journal/diary tracking, and doctor consult tools.

## 🚀 Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Start the development server
```bash
npm run dev
```

### 3. Build for production
```bash
npm run build
```
